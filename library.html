v 0.1.0

A fully usable version of Stallion OS based on more recent OpenSUSE 13.2.Fully UEFI compatible. Even more software is included and Packman repository is pre-installed to allow users to download any software without the need to surf the web. Has GNOME desktop environment installed to ensure the best user experience possible. Has Libreoffice,Rythmbox, Kplayer+codecs included to make sure Stallion OS needs as little configuration as possible. This version is perfect for casual users as it has Wine installed which would allow easy migration from Windows to Stallion OS.
