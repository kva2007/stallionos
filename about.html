About the developer

Stallion OS is made and maintained by one person and the developer welcomes any help and donations from the community.

It was given free programming cloud infrastructure by Microsoft in the form of Microsoft Azure Bizspark subscription.

A brief description

Founded in 2014, Stallion OS is an independent project and an effort to bring a free, open-source and safe operating system, which would not expect the user to be able to use command line interface. Stallion OS started as an effort to understand how software communicates to hardware and then became a full-time hobby. Its aim is to create a stable operating system, which would be free of political or profit-motivated interest. All of us are fed up with the need to configure an operating system when it is installed. With Stallion OS, there is no need to install any software as it runs right out of the box with the minimum pack of software a person might need. This system can be useful for corporate clients as it is easy to install and completely license free so there is no need to hire IT professionals to perform installation. Stallion OS is fully independent and forever free project. It is also a multi-national project
